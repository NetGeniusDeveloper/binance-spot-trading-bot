# Binance Spot Trading Bot — безопасный аналитический crypto scanner

Документация на русском языке для проекта `binance-spot-trading-bot`.

Проект сейчас используется как безопасный аналитический сканер крипторынка и публичных Telegram-каналов. Главный принцип: **аналитика разрешена, автоматическая торговля и создание ордеров запрещены**.

## Commercial version / Коммерческая версия

Этот публичный репозиторий является безопасной демонстрационной версией проекта.

Коммерческая версия развивается отдельно как private multi-exchange AI trading platform.

План коммерческой версии:

- выбор биржи перед настройкой бота;
- Binance как первый адаптер;
- будущие адаптеры для Bybit, OKX, KuCoin, Gate.io и других платформ;
- AI pair discovery;
- liquidity / volatility filters;
- technical scoring;
- news and social risk analysis;
- risk manager;
- backtest laboratory;
- paper trading;
- manager dashboard;
- live trading только после отдельного аудита и ручного включения.

Коммерческая версия не продаётся как программа с гарантированной прибылью.

Это AI-assisted trading framework для аналитики, тестирования, risk-management и контролируемой автоматизации.

Принцип безопасности:

AI предлагает -> risk manager проверяет -> backtest подтверждает -> paper trading тестирует -> live trading только после отдельного аудита.
